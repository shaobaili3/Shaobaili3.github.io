Seminar / Event

セミナー・イベントのご案内

2021.4.6 開催

  • セミナー
  • 受付中

実データで体感！初心者の為のテキストマイニング入門セミナー　～技術解説から活用事例まで～

従来の会場型セミナーをオンライン（web）参加いただける内容に再構成した、
「オンラインハンズオンセミナー」となります。

ベクスト株式会社では、テキストマイニングの世界を「知る・体感する」を目的とした
テキストマイニングの入門セミナーを開催しております。
なかなか触れる機会のない専門ツール（VextMiner）を使い手軽にテキストマイニングを体感頂けます。

特にこんな方へオススメ！

  • テキストマイニングについての基礎知識を得たい方
  • テキストマイニングツールを実際に体験したい方
  • データサイエンティストで社内に大量のテキストをお持ちの方

プログラム内容

＜第１部＞　 テキストマイニングの概要を知る
＜第２部＞　 VextMiner操作体験
＜終了後＞　 質疑応答

1）テキストマイニングの概要を知る（15：00～16：00）

初心者の方も安心の基礎知識講座

親しみやすい例え話を盛り込みながら「テキストマイニングとは？」の概要から最新の活用事例までテキストマイニングの世界を分かりやすく解説いたします。

2）VextMiner操作体験（16：00～17：00）

テキストマイニングツールの操作体験

実際にテキストマイニングツールVextMinerを操作して、実データの分析を体験頂けます。また、活用例としてVextMinerを用いた様々なデータの分析レポートサンプルをご紹介いたします。
※VextMinerでは一部の分析操作を体験頂けます。

※プログラムの時間は進行状況により変更となる可能性がございます。

当日はZoomウェビナー形式となり、Q&Aを機能を利用してその場で疑問ポイントを解決いただけます。また、導入検討やご相談等も別途オンライン上でご対応可能ですので、お気軽にお問合せください。

開催概要

全てのプログラムをWebセミナー（オンラインセミナー）にて実施いたします。

  • 2021年　4月6日（火）
  • 15：00～17：00 (14：50～入室可能)
  • 会場：Web　Zoomウェビナー
  • 参加費無料

社内規定により、調査目的等や個人の方、同業他社様や競合企業様のご参加をお断りさせて頂く場合がございます。ご了承ください。

お申込み方法

以下のフォームよりお申込みください。7営業日以内に受付完了のメールをお送りします。

お申込みフォームへ

PCの推奨環境について（第2部：VextMiner操作体験）

「第2部：VextMiner操作体験」では、講義の中で実際にVextMinerの操作を体験頂くパートがございます。ご自身のPCからインターネット経由でVextMinerへアクセスしツール操作が可能です。ご希望のお客様は以下の推奨環境をご確認の上ご参加ください。もちろん、講義受講のみも可能ですのでご参加用のPCが推奨環境を満たしてない場合もご安心下さい。

推奨環境
CPU Core i5以上 メモリ 4GB以上
OS Windows10 ブラウザ IE 11（標準モード）、Chrome、FireFox、MSEdge
推奨解像度 1600×900 以上 LAN環境 インターネットに接続できる環境が必須です

 

＜ご連絡事項＞
・本セミナー参加前には予めアカウント接続確認をお願いします。
・アカウントはセミナー時のみご利用可能です。貴社情報等のデータを勝手にアップロードした場合、いかなる責任も負いかねますのでご注意ください。
